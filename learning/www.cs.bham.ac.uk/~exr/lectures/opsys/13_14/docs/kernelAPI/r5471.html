clear_inode

Name

clear_inode --  clear an inode

Synopsis

void clear_inode (struct inode * inode);

Arguments

inode

inode to clear

Description

This is called by the filesystem to tell us that the inode is no longer useful. We just terminate it with extreme prejudice.
