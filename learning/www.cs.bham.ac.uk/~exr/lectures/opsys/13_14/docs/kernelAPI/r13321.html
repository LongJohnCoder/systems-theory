uart_get_divisor

Name

uart_get_divisor --  return uart clock divisor

Synopsis

unsigned int uart_get_divisor (struct uart_port * port, unsigned int baud);

Arguments

port

uart_port structure describing the port.

baud

desired baud rate

Description

Calculate the uart clock divisor for the port.
