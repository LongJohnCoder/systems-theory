SEM234 Operating Systems (10cr, Sem 2)

Status: Closed
Staff: Eike Ritter
Objectives
Understanding of basic concepts of operating systems and programming problems in this area.
Module Description
This course gives an introduction into the basic concepts of operating systems like memory management, file systems, threading etc. The course also covers the additional complexities in distributed operating systems.
Delivery
2 lectures a week. There will be an exercise sheet every two weeks which will be covered by a tutorial every two weeks.
Assessment: 2-hour written examination
Faculty: Science
Co-Requisites: None.
Pre-Requisites: None.
Max/Min Course Numbers: unspecified
Key Texts
Abraham Silberschatz and Peter B Galvin, Operating Systems Concepts, Fifth Edition, Addison Wesley, 1998.
A S Tanenbaum, Modern Operating Systems (Prentice-Hall, 1992)
