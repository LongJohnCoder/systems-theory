mca_set_dma_addr

Name

mca_set_dma_addr --  load a 24bit DMA address

Synopsis

void mca_set_dma_addr (unsigned int dmanr, unsigned int a);

Arguments

dmanr

DMA channel

a

24bit bus address

Description

Load the address register in the DMA controller. This has a 24bit limitation (16Mb).
