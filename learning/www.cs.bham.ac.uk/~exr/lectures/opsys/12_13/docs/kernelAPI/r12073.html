blk_queue_free_tags

Name

blk_queue_free_tags --  release tag maintenance info

Synopsis

void blk_queue_free_tags (request_queue_t * q);

Arguments

q

the request queue for the device

Notes

blk_cleanup_queue will take care of calling this function, if tagging has been used. So there's usually no need to call this directly, unless tagging is just being disabled but the queue remains in function.
